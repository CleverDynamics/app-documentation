    Document Promotion Codes

    It also adds two icons to the Navigate Tab on the Ribbon, Document Promotion Codes and Promotion Summary.

    The Document Promotion Codes action allows entry of previously created promotion codes, which can then be applied using the Apply Promotion button.

    To use the promotion codes, go to the ribbon on the Navigate Tab and choose Document Promotion Codes. Here, enter the codes in the edit screen.

    Once the code is entered, click to move off the box, then click the cross to close that edit screen. The code is now stored on the sales document and is counted as having been used.

    Attempting to use that same code again in a sales document will produce an error message, as the code does not pass validation checks, this prevents further progress, to continue the user would need to use a different code instead.
